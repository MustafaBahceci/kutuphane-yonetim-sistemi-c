# kutuphane-yonetim-sistemi-c  
Bu proje, C programlama dili kullanılarak geliştirilen **basit bir kütüphane yönetim sistemi**dir.  
Konsol tabanlı olarak çalışır ve kullanıcıdan alınan bilgilerle kitap kayıtlarını yönetir.

## Özellikler
- Kitap ekleme
- Tüm kitapları listeleme
- Belirli bir yazara ait kitapları gösterme
- Kütüphanedeki toplam kitap sayısını gösterme
